windows user switch underline

win11 has a colored underline under the password field when switching users. it's supposed to have a color of the selected user's accent color, but you can break it by:
  1. logging out
  2. clicking on other user
  3. clicking on the user you logged out from
now it is colored with the other user's accent color.

removed due to insufficient impact on user experience
